### 1.2 python一些基础语法

```
1. print()  打印数据  print('Hello World') 
2. type()  使用type函数可以查看一个变量的类型
3. input()  字符串变量 = input('提示信息：：：')
4. int(x)  将x转换为一个整数
5. float(x)  将x转换为一个浮点数
6. random.randing(a,b) a<x<b
```
